# Arshrozy Printshop Website

A premium, responsive website for a professional printing business in Kenya.

## Features
- **Product Catalog**: Business keys, Banners, Marketing Materials, Apparel, and more.
- **Pricing Calculator**: Interactive price estimation for products.
- **WhatsApp Integration**: Direct ordering via WhatsApp with pre-filled details.
- **Responsive Design**: Mobile-friendly layout using modern CSS.

## Structure
- `index.html`: Main landing page.
- `style.css`: Main stylesheet with premium design system.
- `script.js`: Logic for pricing calculators and WhatsApp integration.
- `images/`: Directory for assets.
- `*.html`: Individual product category pages (e.g., `apparel.html`, `business-cards.html`).

## Setup
1.  Clone the repository.
2.  Ensure `style.css` and `images/` are in the root directory.
3.  Open `index.html` in any web browser.

## Customization
- **Pricing**: Update the `PRICES` object in `script.js`.
- **Phone Number**: Update `PHONE_NUMBER` constant in `script.js` to change the WhatsApp recipient.
- **Colors**: Edit CSS variables in `:root` of `style.css`.
